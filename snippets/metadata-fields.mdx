### System Metadata Fields

These fields are automatically populated by the system and should not be included in your submission:

| Field | Description |
|-------|-------------|
| `CREATED_TIMESTAMP` | Date/time record was created |
| `CREATED_BY` | User/system that created the record |
| `UPDATED_TIMESTAMP` | Date/time record was last updated |
| `UPDATED_BY` | User/system that last updated the record |
