## Overview

County codes use the standard 3-digit Federal Information Processing Standard (FIPS) codes for Colorado counties. These codes are used in fields like `COUNTY_CODE` to identify client residence or service delivery location.

## Special Codes

| Code | Description |
|------|-------------|
| `999` | Unknown / Not Reported |
| `000` | Out of State |

## Usage Examples

```csv
# Denver County
COUNTY_CODE
031

# Boulder County
COUNTY_CODE
013

# Unknown
COUNTY_CODE
999
```

## Notes

- All codes are 3 digits with leading zeros
- Use `000` for clients residing outside Colorado
- Use `999` only when county cannot be determined
