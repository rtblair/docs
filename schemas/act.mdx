## Overview

The ACT (Assertive Community Treatment) schema captures data for intensive community-based mental health treatment programs. ACT teams provide comprehensive services to individuals with severe mental illness who require intensive support to live successfully in the community.

<Note>
This schema is **only required** for clients enrolled in ACT programs. Submit ACT records for each reporting period during enrollment.
</Note>

## 1. Schema Reference

### Required Fields

| BHA Column Name | Field Name | Format | Valid Values | Constraints | Example |
|-----------------|------------|--------|--------------|-------------|---------|
| `RECORD_TYPE` | Record Type | `x(3)` | `ACT` | Must be exact | `ACT` |
| `TRANSACTION_TYPE` | Transaction Type | `x(1)` | `A`, `U`, `R` | See [Transaction Types](/getting-started/transaction-types) | `A` |
| `BHASO_CLIENT_IDENTIFIER` | Client Identifier | `x(20)` | Alphanumeric | Must match admission | `CLT-000001` |
| `BHASO_CASE_IDENTIFIER` | Case Identifier | `x(20)` | Alphanumeric | Must exist in ADMISSION | `CSE-000001` |
| `EFFECTIVE_DATE` | Effective Date | `date` | YYYY-MM-DD | Cannot be future | `2024-01-31` |
| `REPORTING_PERIOD_START` | Period Start | `date` | YYYY-MM-DD | First day of month | `2024-01-01` |
| `REPORTING_PERIOD_END` | Period End | `date` | YYYY-MM-DD | Last day of month | `2024-01-31` |
| `ACT_TEAM_ID` | ACT Team ID | `x(10)` | Alphanumeric | Valid team identifier | `ACT-001` |
| `ENROLLMENT_STATUS` | Enrollment Status | `x(1)` | `1`, `2`, `3` | See codes below | `1` |

### Enrollment Status Codes

| Code | Description |
|------|-------------|
| `1` | Active - Full ACT services |
| `2` | Step-down - Transitioning to lower intensity |
| `3` | Graduated - Successfully transitioned out |

### Service Delivery Fields (Required)

| BHA Column Name | Field Name | Format | Valid Values | Constraints | Example |
|-----------------|------------|--------|--------------|-------------|---------|
| `FACE_TO_FACE_CONTACTS` | Face-to-Face Contacts | `9(3)` | 0-999 | Monthly total | `12` |
| `PHONE_CONTACTS` | Phone Contacts | `9(3)` | 0-999 | Monthly total | `8` |
| `COMMUNITY_CONTACTS` | Community Contacts | `9(3)` | 0-999 | In community (not office) | `10` |
| `CRISIS_CONTACTS` | Crisis Contacts | `9(3)` | 0-999 | Emergency/crisis | `2` |

| BHA Column Name | Field Name | Format | Valid Values | Constraints | Example |
|-----------------|------------|--------|--------------|-------------|---------|

| `JAIL_DAYS` | Jail/Prison Days | `9(3)` | 0-31 | During period | `0` |
| `HOMELESS_DAYS` | Homeless Days | `9(3)` | 0-31 | During period | `0` |

- Row 1: `ACT-003` - REPORTING_PERIOD_START must be first day of month
- Row 2: `XFD-001` - HOSPITALIZATION_DAYS required when PSYCHIATRIC_HOSPITALIZATION = 1

## 3. Validation Rules

### Error Codes

| Code | Message | Resolution |
|------|---------|------------|
| `ACT-001` | ACT_TEAM_ID is required | Provide valid ACT team identifier |
| `ACT-002` | ENROLLMENT_STATUS is required | Provide status code 1, 2, or 3 |
| `ACT-003` | REPORTING_PERIOD_START must be first day of month | Use first day of reporting month |
| `ACT-004` | REPORTING_PERIOD_END must be last day of month | Use last day of reporting month |
| `ACT-005` | Admission not found | Submit ADMISSION record first |
| `ACT-006` | Duplicate reporting period | Only one ACT record per month |

### Cross-Field Validation

| Rule | Fields | Constraint |
|------|--------|------------|
| Hospitalization | `PSYCHIATRIC_HOSPITALIZATION`, `HOSPITALIZATION_DAYS` | If hospitalized, days required (>0) |
| Period Dates | `REPORTING_PERIOD_START`, `REPORTING_PERIOD_END` | Must be same calendar month |
| Contact Totals | `COMMUNITY_CONTACTS`, `FACE_TO_FACE_CONTACTS` | Community â‰¤ Face-to-face |

## 6. Update Semantics

### Primary Key

`BHASO_CASE_IDENTIFIER` + `REPORTING_PERIOD_START`

### Transaction Types

| Type | Code | Behavior |
|------|------|----------|
| Add | `A` | Creates new monthly ACT record |
| Update | `U` | Updates existing period record |
| Replace | `R` | Overwrites period record |
