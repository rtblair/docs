## Overview

The SC BABY schema captures birth and baby outcome data for the Special Connections (SC) program, which serves pregnant and postpartum women with substance use disorders. This record documents outcomes for babies born to program participants.

<Note>
This schema is **only required** for Special Connections program participants who give birth during enrollment. Submit one BAB record per baby born.
</Note>

## 3. Validation Rules

### Error Codes

| Code | Message | Resolution |
|------|---------|------------|
| `BAB-001` | BABY_IDENTIFIER is required | Provide unique baby identifier |
| `BAB-002` | BIRTH_DATE is required | Provide birth date |
| `BAB-003` | GESTATIONAL_AGE must be 20-45 weeks | Check gestational age |
| `BAB-004` | BIRTH_WEIGHT must be 300-7000 grams | Check birth weight |
| `BAB-005` | Admission not found | Submit ADMISSION record first |
| `BAB-006` | Duplicate BABY_IDENTIFIER | Each baby needs unique ID |

### Cross-Field Validation

| Rule | Fields | Constraint |
|------|--------|------------|
| APGAR Scores | `LIVE_BIRTH`, `APGAR_1MIN`, `APGAR_5MIN` | Required if live birth |
| NICU Days | `NICU_ADMISSION`, `NICU_DAYS` | Required if NICU admission |
| NAS Treatment | `NAS_DIAGNOSIS`, `NAS_TREATMENT` | Required if NAS diagnosed |
| Multiple Birth | `MULTIPLE_BIRTH`, `BIRTH_ORDER` | Birth order required if multiple |
| Death | `LIVE_BIRTH`, `DEATH_DATE` | Required if stillbirth |
